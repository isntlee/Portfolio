Projects

Sagacity

Python JavaScript Flask MongoDB
jQuery GoogleMaps API Heroku Materialize

For all of Dublin's local history collections none are particularly interactive or user-friendly. This site is an effort to correct that, the project is now a public collection of city tales.

Visitors can add, edit, and maintain their own stories, while also touring the city and its history through the GoogleMaps API. This was built using the Flask micro-web framework and hosted with Heroku.

There's a host of search, sort, and filtering options available so you can browse the collection with confidence.

Jellett Gallery

Python JavaScript Django PostgreSQL
Stripe API Heroku jQuery Bootstrap

This is a gallery that exhibits the work of young, upcoming female artists in Ireland. Built using the Django framework and hosted on Heroku.

Fundamentally this is an e-commerce site, using its simple and intuitive UI visitors can browse the gallery but also save, bid and purchase these pieces.

Transactions are conducted through the Stripe API, the production database chosen was PostgreSQL.

counterCarbon

JavaScript HTML CSS
jQuery Highcharts GIMP Bootstrap

The key ambition of the project was to make this site better than the broad choice of carbon trackers, which are frustratingly detailed, often requiring precise inputs (.e.g. rate of heating oil use).

Graphing/displaying data is handled by the Highcharts library, user interaction through vanilla Javascript

The site is designed as an intuitive, user-friendly and encouraging introduction to a more carbon-conscious world.
